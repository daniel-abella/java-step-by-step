Ciclo de vida de um Servlet

Quando uma requisição do tipo http://localhost:8080/pa25/ciclo é atendida, uma entrada correspondente é registrada no arquivo log.txt. De fato, quando o servlet é iniciado, um registro correspondente é feito neste arquivo; quando o servlet é terminado, um registro correspondente também é realizado e assim por diante. Por meio deste arquivo, portanto, é possível "acompanhar" o ciclo de vida (eventos relevantes) e as requisições correspondentes a um servlet.

Sugestão para experimentar este servlet

Inicie o tomcat, faça algumas requisições a URL fornecida acima, ou simplesmente clique aqui (esta mesma página será reapresentada). Em seguida, interrompa a execução do Tomcat. Posteriormente, consulte o conteúdo do arquivo log.txt gerado no diretório de contexto da aplicação. Ou seja, o caminho completo para o arquivo deve ser $CATALINA_HOME/webapps/pa25/log.txt.
Alternativamente, clique aqui para obter o conteúdo do arquivo log.txt.
